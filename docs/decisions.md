# Решения

## 2026-02-09 — Размещение приложения
**Решение:** разместить Next.js приложение в корне репозитория.
**Причина:** на старте быстрее упростить dev-опыт без монорепо.

## 2026-02-09 — Контекст бренда школы
**Решение:** базовый механизм брендинга планируется через маршрут `/s/[workspaceSlug]`.
**Причина:** минимальная реализация мульти-тенантности без отдельного домена.

## 2026-02-09 — Провайдер видеозвонков
**Решение:** по умолчанию использовать Jitsi (meeting_provider = `jitsi`).
**Причина:** быстрый MVP с встраиваемым iframe.
